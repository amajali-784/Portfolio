# Ankush Majalikar Portfolio OS

3D animated AI/ML portfolio built with Next.js for Vercel deployment.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Check

```bash
npm run build
npm run start
```

## Deploy To Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Keep the default framework preset as Next.js.
4. Deploy.
